### Datasets

The analysis uses one dataset: 

- [National Risk Index Counties feature layer](https://www.arcgis.com/home/item.html?id=39485e8035d446a5bff03259508ae355) A csv file containing county-level data on the risk index, expected annual loss, social vulnerability, and community resilience for all US states and territories.
